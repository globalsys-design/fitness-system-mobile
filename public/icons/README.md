# PWA Icons

Place the following icon files here:

- `icon-192.png` — 192x192 pixels (required for PWA install)
- `icon-512.png` — 512x512 pixels (required for PWA splash screen)
- `icon-512-maskable.png` — 512x512 pixels with safe zone (for Android adaptive icons)

## How to generate icons

1. Design your icon (at least 512x512 pixels)
2. Use [PWA Builder](https://www.pwabuilder.com/imageGenerator) or [Maskable.app](https://maskable.app/editor) to generate all sizes
3. Place the generated files in this directory

## Color Reference
- Primary color: `#00b5b0` (cyan — converted from oklch(0.6566 0.1120 194.7713))
- Background: `#ffffff` (white)
